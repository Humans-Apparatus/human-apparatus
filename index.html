01 / About

We believe in software that serves its user, not the other way around. As interfaces grow noisier and more extractive, there is a quiet case for text-native tools — fast, composable, scriptable, and built in the open.

Human Apparatus is a registered non-profit developing a suite of TUI-first applications under a single coherent philosophy: the terminal is not a step backward — it is the most honest interface ever made.

01.
TUI First Every product works completely in a terminal before a GUI is ever considered.
02.
Text Native Data lives in plain text. Configs are readable. Nothing is locked in a binary blob.
03.
Composable Tools pipe into each other. No monoliths. Unix philosophy, applied consistently.
04.
Permanently Open Source All code is free software. No open-core, no cloud-only features, no bait-and-switch.
05.
Human's, not AI-first Not anti-AI — simply clear that tools exist to augment humans, not replace them.

04 / Contribute

Code

Fork any repo, open a PR, write a test. We use conventional commits and keep changelogs honest. All PRs are reviewed by a human. Start at github.com/human-apparatus.

Documentation

Docs live alongside code in each repo. If something is unclear, fix it in a PR. Plain Markdown, no fancy toolchains required.

Donate

We're a non-profit under registration. Donations will be accepted once formalized. Watch this page for updates.

Spread the Word

Star repos. Write a blog post. Mention us in a talk. The most effective marketing for free software is people who genuinely use it.
